# Cracking-Squishy-Ball

**Three.js 기반 왁스 코팅 클레이볼 부수기 시뮬레이터 웹 애플리케이션**

단일 HTML5 파일 내 Three.js를 활용하여 구축된 3D 인터랙티브 웹 애플리케이션으로 '왁스 코팅 클레이볼'의 경험을 디지털 환경에서 부분적으로 구현하였습니다.  
실제 제품과 동일한 물리 반응을 완벽하게 재현하기보다는, 웹 환경에서 사용자가 투명한 막으로 감싸진 왁스 층을 부수고 내부 클레이를 변형시키는 과정을 시각적·청각적으로 경험할 수 있도록 설계되었습니다.

---

## 개요

**Cracking-Squishy-Ball** 프로젝트는 브라우저에서 바로 실행되는 3D 인터랙티브 웹 애플리케이션입니다.  
Three.js와 Web Audio API를 활용하여 3D 렌더링, 보로노이 분할 로직, 실시간 사운드 합성 및 사용자 상호작용을 구현했습니다.  
사용자가 왁스 층을 부수고 내부 클레이를 변형시키는 과정을 직접 제어할 수 있고, 시각적·청각적으로 몰입감 높은 시뮬레이터 경험을 제공합니다.

---

## 기술 스택

* **HTML5 / CSS3** - 디바이스 환경(모바일, 태블릿, PC)에 맞춘 반응형 레이아웃 및 UI 구성
* **JavaScript (ES6+)** - 보로노이 분할, 쿼터니언 제어, 변형 로직 및 이벤트 핸들링
* **Three.js** - 3D 렌더링(WebGLRenderer, ACESFilmicToneMapping) 및 시뮬레이션 환경 구축
* **Web Audio API** - 오디오 버퍼 소스 및 로우패스 필터, 복수의 오실레이터를 조합한 실시간 사운드 합성
* **OrbitControls** - 빈 공간 드래그를 통한 카메라 회전 및 시점 탐색 제어

---

## 주요 기능

- 물리 기반 부수기: 사용자 터치 입력 시 `fractureAt` 함수가 작동하여 보로노이 분할 방식으로 왁스 메쉬를 쪼개며, 파편의 이동 속도는 매 프레임마다 감쇠 처리를 거쳐 안정적으로 안착
- 클레이 변형(Grind): 왁스가 부서져 제거된 부위를 문지르면 `grindChunk` 및 `paintP` 함수가 작동하여 클레이 메쉬의 정점을 함몰
- 자동 완료 시스템: 전체 부수기 진행률이 95%에 도달하면 `updateHud` 로직에 의해 남은 미세 조각을 강제로 제거하여 100% 완료 상태로 자동 전환
- 실시간 오디오 합성: Web Audio API의 오실레이터를 활용하여 파괴 강도 및 메쉬 분리 속도에 따라 즉석에서 사운드를 합성하여 청각적 피드백을 제공
- 시점 탐색: `OrbitControls`를 사용하여 마우스나 터치 드래그로 3D 공간을 자유롭게 회전하며 부수는 과정을 다각도에서 관찰
